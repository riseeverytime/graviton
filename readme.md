## ✨ Features

- 🧬 Git integration
- 🟢 NPM scripts integration
- 🛢 Workspaces
- 💅 Modern UI
- 🎨 Themes
- 📦 Plugins
- 🌠 Autocomplete for some languages
- 🗺 Built-in image previewer
- 🏭 A Plugin and Themes store
- 😇 More respectful on RAM than other alternatives
- 💆‍♀️ No telemetry
- 📝 Multi line editing

## 💻🖥 Available platforms

**Windows**

- `Executable (.exe, 64bits)`

**Linux**

- `Appimage (.appimage)`
- `Debian (.deb)`
- `CentOS (.rpm)`

**MacOS**

- `DMG file (.dmg)`
