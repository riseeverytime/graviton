This is a web-based frontend implementation

| Folder       | Description        |
|--------------|--------------------|
| components   | General components |
| editors      | Built-in editors   |
| modules      | API utilities      |
| panels       | Built-in panels    |
| tabs         | Built-in tabs      |
| types        | General types      |
| utils        | General utils      |

Install dependencies:
```
yarn
```

Run:
```
yarn dev 
```
